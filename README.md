# Leaflet.glify
web gl renderer plugin for leaflet


inspired by http://bl.ocks.org/Sumbera/c6fed35c377a46ff74c3 & need


## Polygon Usage
```javascript
L.glify.shapes({
  map: map,
  data: geoJson
});
```

## Points Usage
```javascript
L.glify.points({
  map: map,
  data: points
});
```
